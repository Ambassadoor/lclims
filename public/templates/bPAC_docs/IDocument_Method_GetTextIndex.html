LONG bpac::IDocument::GetTextIndex (BSTR fieldName)
 

Acquires the text index

Returns the index of the text having the specified field name (template text).
When specifying the field name on the template, the text merely need to be entered.
If there are multiple text lines having the same field name (template text) in one template file, the index for the frontmost text line is returned.

Arguments:
fieldName  Field name (template text) of the text for which index is to be acquired
Returned value:
Text index
