Sample templates included with SDK

The print area and printer for the templates in the b-PAC SDK template folder (normally C:\Program Files\Brother bPAC3 SDK\Templates) are have been set for the PT-9500PC.


If output is possible with P/M/S-series printer drivers other than the PT-9500PC, we recommend changing the templates according to the following procedure.

  1. In P-touch Editor, open a layout file (*.lbx or *.lbl) from the template folder mentioned above.
     
  2. From the menu, select [File] – [Printer Setup...].
     
  3. Select the printer that will be used to output the data.
     
  4. Adjust the layout if any part extends out of the print area.
     
  5. Save the layout file by overwriting the original.
