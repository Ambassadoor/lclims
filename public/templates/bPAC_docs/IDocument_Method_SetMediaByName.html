VARIANT_BOOL bpac::IDocument::SetMediaByName (BSTR mediaName, VARIANT_BOOL fitPage)
 

Sets the media to be used, by media name

Changes are made to the media of the media name specified in mediaName.

Arguments:
mediaName  Media name.
Specify a media name obtainable from IPrinter::GetSupportedMediaNames().
fitPage  As the layout changes due to media changes, specify whether to adjust the size and position of the object in the template in conformity with the changes.
Adjustment is made if VARIANT_TRUE is specified, and no adjustment is made if VARIANT_FALSE is specified.
Returned value:
Success or Failure
Reference:
MediaName
