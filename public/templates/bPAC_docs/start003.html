Using b-PAC on .NET Framework

 When using b-PAC with managed code in the development environment of Visual Studio .NET2003 or later, "Add Reference" is required.
Specify "Brother b-PAC 3.4 Type Library"  as the name of the component to be used.

 For the procedure to "Add References", refer to the following pages.

https://docs.microsoft.com/en-us/visualstudio/ide/managing-references-in-a-project?view=vs-2019
