LONG bpac::IObjects::GetIndexByName (BSTR name, LONG indexBgn)
 

Acquires the object index

Searches the objects in the sequence of the index using the object name specified in name, and acquires the index of the first object found.

Arguments:
name  Object name for acquiring the index.
indexBgn  Start position of object search.
Starts the object search starting from the specified index.
Returned value:
Searches and returns the index of the object found.
Returns -1 if not even one object is found with the specified name.
