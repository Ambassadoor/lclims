VARIANT_BOOL bpac::IObject::SetSelection (LONG start, LONG end)
 

Selects a character substring

Selects the character substring specified by start and end.
In the initial state, the whole character string is selected.
This method is applicable to text objects only.

Arguments:
start  Start position of selection.
If a value lower than 0 is specified, it becomes 0.
end  End position of selection.
If a value higher than the maintained character count is specified, the selection is until the tail end of the character string.
Returned value:
Success or Failure
Reference:
SelectionStart, SelectionEnd
