Using b-PAC on VBA

When using b-PAC in Excel VBA or Access VBA, you should set a reference to b-PAC.
Specify "Brother b-PAC 3.4 Type Library"  as the name of the component to be used.

For the procedure to set a reference, refer to the following pages.

http://msdn.microsoft.com/en-us/library/aa164510%28office.10%29.aspx
