Layout units

The units on the layout used in b-PAC are 1440dpi units.
Refer to the description below for the conversion of each unit. 

Conversion between logical units and mm

  • Logical units => mm
     
    Y = X * 25.4 / 1440
  • mm => logical unit
     
    Y = (X * 1440) * 10 / 254

Conversion between logical units and inches

  • Logical units => inches
     
    Y = X / 1440
  • Inches => logical units
     
    Y = X * 1440
