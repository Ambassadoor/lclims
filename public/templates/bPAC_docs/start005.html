b-PAC 3.4 SDK Configuration

When the Brother b-PAC SDK is installed, the files are copied to the following folders.
  • Application folder (normally C:\Program Files\Brother bPAC3 SDK\)

    .\Doc
            This document, the user license agreement, and a list of redistributable samples

    .\Redist
            Merge Module for redistribution (bPAC3.msm/bPAC3_64.msm)

    .\Samples
            Respective VBA, VBNET, VBS, VC, VCS sample projects

    .\Templates
            Template files for use with the sample application

  • Common files folder (normally C:\Program Files\Common Files\Brother\b-PAC)

    Component and type library

Note!
The sample applications copied to the Sample folder assume that the templates are located in the folder at C:\Program Files\Brother bPAC3 SDK\Templates.
If the Brother b-PAC SDK is installed into a folder other than C:\Program Files\Brother bPAC3 SDK\, copy all files in the installed Templates folder to C:\Program Files\Brother bPAC3 SDK\Templates.
