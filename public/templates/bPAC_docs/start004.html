Program Redistribution

 When redistributing programs created using the Brother b-PAC 3.4 SDK, the following components and files must be installed on the client computer where the redistributed program is installed.
 The Brother b-PAC SDK itself, cannot be redistributed.

  1. b-PAC component

    a. To install only the component onto a client PC
      Use the b-PAC3.4 client component downloaded from the b-PAC3.4 download site.
      For detailed operating instructions on the client component, refer to the b-PAC3.4 download site.

    b. If an installer development environment such as Visual Studio and InstallShield is being used to create an installer for the created application
      Add the following files to the installer project.

      For the 32-bit version of b-PAC)
      Åï Microsoft_VC142_MFC_x86.msm, Microsoft_VC142_CRT_x86.msm

      For the 64-bit version of b-PAC)
      Åï Microsoft_VC142_MFC_x64.msm, Microsoft_VC142_CRT_x64.msm, Microsoft_VC110_CRT_x64.msm

      Åï "bPAC3.msm/bPAC3_64.msm" from the Redist folder of the application folder
      With "bPAC3.msm/bPAC3_64.msm", the component can be used with multiple projects, and the registry and type library of the component are also registered.
      For details on the procedure to add it, refer to the following page.
        http://msdn2.microsoft.com/en-us/library/8x727h8b(VS.80).aspx

    c. If a merge module is not supported in the development environment being used to create an installer for your application
      Use "bPAC3CCISetup.msi/bPAC3_64CCISetup.msi" from the Redist folder in the application folder.

  2. Printer driver of target printer
    To set up a target printer, run the driver installer from the P-touch Editor CD-ROM, or download the correct driver from the Brother support website (https://support.brother.com/).
    If you have already installed the printer driver, you do not need to install it again.

  3. Template files  
    Since b-PAC refers to the templates, the templates created using P-touch Editor are needed.
    The templates can even be stored on a file sharing server.
    In addition, if the target driver is installed and the template files are available, P-touch Editor is not needed to run b-PAC.
    For details, refer to "What are template files?".
