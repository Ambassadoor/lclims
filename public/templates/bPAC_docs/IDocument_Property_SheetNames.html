VARIANT bpac::IDocument::SheetNames [get]
 

Sheet name enumerator (VT_VARIANT | VT_ARRAY)

Acquires the sheet names of all sheets in the template through character string array.

       //
       // C# sample
       //
       object[] sheetNames = doc.SheetNames as object[];
       foreach (object sheetName in sheetNames)
       {
           doc.CurrentSheet = strSheetName as string;
       }
