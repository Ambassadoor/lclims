Checking a template

 After creating the template file, open it and check that it is displayed correctly.
 Refer to the following checklist when checking the template.


Overall layout

  • With label templates, do the objects have enough extra space within the print area?
  • With stamp templates, do all objects fit within the print area?
    (However, special layouts containing many Arrange Text objects and clip art are excluded.)
  • Have all unnecessary objects been removed from the layout?
    (Check by holding down the Tab key to select each object.)
  • Is any necessary character formatting lost when the text is displayed using the [Text Box] command?
    (when character formatting is applied)
Text
  • Except when the Fit option is selected, does the font of all text appear in the original height-to-width ratio?
    (especially when the Auto option is selected)
  • If the Clip option is selected, is the text object located so that it will not hide other objects (lines, etc.), does it provide for characters being added, and is it adjusted to a size with enough extra space?
  • Have no unnecessary spaces been added to the text? Instead, is it not possible to adjust the character spacing? 
  • Is any necessary character formatting lost when the text is displayed using the [Text Box] command?
    (when character formatting is applied)
Barcode
  • Is there enough space around the barcode object to prevent it from extending beyond the print area if the length changes due to the barcode data or from covering other objects on either side (when the barcode is horizontally positioned)?

Others
  • Display the layout at a zoom ratio of 1:1. Check how the objects overlap. Be particularly careful when grouping rectangles and straight lines and when drawing lines. In addition, if a bitmap is used in the template, check how it will be reproduced.
  • Display the layout at a zoom ratio of 100%. Are there no missing parts in the objects and is the balance of the objects the same as when the layout is displayed normally?
