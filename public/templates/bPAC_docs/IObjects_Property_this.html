IObject bpac::IObjects::this [LONG index] [get]
 

Indexer(refer to the usage example for the usage method)

Acquires the object specified in index.

Arguments:
index  Object index to be acquired
Returned value:
Object specified in index
              //
            // C# sample
              //
            bpac.IObjects objs = doc.Objects;
            for (int i = 0; i < objs.Count; ++i)
            {
                objs[i].Name;
            }
Exception:
  If an invalid index is specified in index, an exception is thrown.
