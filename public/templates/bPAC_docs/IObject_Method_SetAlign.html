VARIANT_BOOL bpac::IObject::SetAlign (LONG horizontal, LONG vertical)
 

Sets the object alignment

Arguments:
horizontal  Horizontal alignment
Value Name Description
0 Left Left-aligned
1 Center Centered
2 Right Right-aligned
3 Justify Justify*
4 Equal Length Equal Length*
* Can be specified only when the ObjectType is bobText.

vertical  Vertical alignment
Value Name Description
0 Top Top-aligned
1 Middle Centered
2 Bottom Bottom-aligned
Returned value:
Success or Failure
Reference:
HorizontalAlign, VerticalAlign
