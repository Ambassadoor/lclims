VARIANT bpac::IDocument::GetImageData (ExportType type, LONG width, LONG height)
 

Acquires a preview sample image (VT_UI1 | VT_ARRAY)

Acquires image data of the specified size.
This method cannot be used by VBScript and JScript.

Arguments:
type  Type of image to be acquired.
Only bexBmp can be specified.
width  Horizontal size (pixel) of the image to be acquired.
If 0 is specified, it becomes a size that maintains the aspect ratio based on height.
height  If the vertical size (pixel) of the image to be acquired is specified as 0, it becomes a size that maintains the aspect ratio based on width.
Returned value:
Sample image's binary data (VT_UI1 | VT_ARRAY)
