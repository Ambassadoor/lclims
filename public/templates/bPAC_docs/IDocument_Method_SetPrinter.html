VARIANT_BOOL bpac::IDocument::SetPrinter (BSTR printerName, VARIANT_BOOL fitPage)
 

Printer settings

Arguments:
printerName  The specified printer name.
Specify a printer name acquired through IPrinter::GetInstalledPrinters().
fitPage  As the layout changes due to media changes, specify whether to adjust the size and position of the object in the template in conformity with the changes.
Adjustment is made if VARIANT_TRUE is specified, and no adjustment is made if VARIANT_FALSE is specified.
Returned value:
Success or Failure
Reference:
Printer
