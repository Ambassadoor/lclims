VARIANT_BOOL bpac::IDocument::DoPrint (LONG dwOption, BSTR szOption)
 

Prints the document
(This function has been retained to maintain compatibility)

Arguments:
dwOption  Print option.
Refer to bpac::PrintOptionConstants.
szOption  Print parameters.
Specify the parameters for during printing, using a ',' (comma)-separated character string.
Index Name Description
0 Name of document to be printed Specify the document name to be displayed in the corresponding print queue.
If not specified or '0' is specified, the template name is displayed in the print queue.
1 Number of copies Specify the number of copies.
If not specified, the number of copies is normally 1.
For stamps, the setting for the number of copies is invalid.
ex."MyDoc, 2" : Besides displaying "MyDoc" in the print queue, 2 copies are printed.
Returned value:
Success or Failure
Reference:
StartPrint(), PrintOut(), EndPrint()
