Sample Application

The sample application assumes that the templates are located in the folder C:\Program Files\Brother bPAC3 SDK\Templates.
If the Brother b-PAC SDK is copied to a different folder, copy all files in the installed Templates folder to the folder at  C:\Program Files\Brother bPAC SDK\Templates.

In addition, this limitation only applies to the sample application, and not to the b-PAC component.

Refer to the appropriate information for the development environment being used.

  • Visual C++® (VC folder)

  • Visual Basic® for Application (VBA folder)

  • VBScript (for Windows Scripting Host) (VBS folder)

  • VBScript (for Internet Explorer) (VBS folder)

  • JScript (for Internet Explorer) (jscript folder)

  • Visual Basic® .NET (VBNET Folder)

  • Visual C#® (VCS folder)

f the application does not run correctly, refer to "Troubleshooting".
