VARIANT_BOOL bpac::IDocument::SetBarcodeData (LONG index, BSTR barcodeData)
 

Sets the barcode data

Sets/updates the barcode data for the specified barcode.

Arguments:
index  Barcode data index
barcodeData  Character string representing the barcode
Returned value:
Success or Failure
index:
For the rules on the indexing of barcode objects, refer to Barcode Object.

barcodeData:
only when the barcode is CODE128/EAN128
Specify "\a" if FNC1 is to be inserted into the data.
For example, for setting "1234"+FNC1+"567"+FNC1, express it as shown below. 
ObjDoc.SetBarcodeData 0, "1234\a567\a" (For VBS)
For VC, it becomes "\\a".

The substitution rules for the control codes are as shown below.

FNC1 "\a"
FNC2 "\b"
FNC3 "\c"
FNC4 "\d"
NUL "\0" VT"\B" SYN "\M"
SOH "\1" FF"\C" ETB "\N"
STX "\2" CR"\D" CAN "\O"
ETX "\3" SO"\E" EM"\P"
EOT "\4" SI"\F" SUB "\Q"
ENQ "\5" DLE "\G" ESC "\R"
ACK "\6" DC1 "\H" FS"\S"
BEL "\7" DC2 "\I" GS"\T"
BS "\8" DC3 "\J" RS"\U"
HT "\9" DC4 "\K" US"\V"
LF "\A" NAC "\L" DEL "\W"
