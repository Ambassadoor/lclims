VARIANT_BOOL bpac::IObject::SetPosition (LONG x, LONG y, LONG width, LONG height)
 

Sets the object's position and size

Sets the object's position and size.
The unit is 1440dpi.
Depending on the specified value, the position may shift to that different from the specified value.

Arguments:
x  Upper-left horizontal position
y  Upper-left vertical position
width  Object's horizontal width
height  Object's vertical width
Returned value:
Success or Failure
Reference:
X, Y, Width, Height
