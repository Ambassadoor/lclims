Requirements for creating programs

  • Brother b-PAC 3.4 SDK (including template files used with sample applications)
    For details on the SDK configuration, refer to "b-PAC 3.4 Configuration".
     
  • Printer driver of target printer
    To set up a target printer, run the driver installer from the P-touch Editor CD-ROM, or download the correct driver from the Brother support website (https://support.brother.com/) .
    If you have already installed the printer driver, you do not need to install it again.
     
  • P-touch Editor
    This is necessary for creating new templates or editing some templates provided with the b-PAC 3.4 SDK.
     (This is not needed if the templates provided with the b-PAC 3.4 SDK are simply to be used as they are.)
    Use P-touch Editor if you want to edit the templates provided with b-PAC 3.4 SDK, or to create new ones. To install P-touch Editor, run the installer from the P-touch Editor CD-ROM, or download the correct installer from the Brother support website (https://support.brother.com/).
    For details on creating templates, refer to "What are template files?".

In addition, sending a created program to a different computer where it can be used to create labels is called "redistribution".
For details on redistribution, refer to "Program Redistribution".
