VARIANT_BOOL bpac::IDocument::Open (BSTR filePath)
 

Opens the template file

Arguments:
filePath  Path (UNC) or URL to the template file
Can be specified as
\\your_server\your_folder\your_file
for a file on a file-sharing server
Can be specified as
http://your_server/your_file
for a template file URL
Returned value:
Success or Failure
Reference:
Close()

- Sample for obtaining the path from environment variable and opening it
