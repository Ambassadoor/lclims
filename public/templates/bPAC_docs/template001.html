What are template files?

In order to use b-PAC, template files must be created using P-touch Editor Version 4 or more.

With P-touch Editor, it is not necessary to save template files in a special format.
They can be saved as a normal layout file (*.lbl, *.lbx).

b-PAC outputs the template file (=layout file) with the same size (width and length) that was set using the printer driver that was set.
Therefore, regardless of the current printer settings, you can output using the desired driver and label size.

   (*Note) Also refer to the " Limitations on b-PAC templates".
