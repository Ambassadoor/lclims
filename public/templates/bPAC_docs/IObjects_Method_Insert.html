IObject bpac::IObjects::Insert (LONG index, ObjectType type, LONG x, LONG y,
                                                   LONG width, LONG height, VARIANT option)
 

Inserts a new object

Arguments:
index  Index of object to be inserted.
If specified to be higher than the value acquired by Count, it is inserted at the tail end.
type  Object type.
At present, only bobText is supported.
x  Object's upper-left horizontal position.
y  Object's upper-left vertical position.
width  Object's width.
height  Object's height.
option  Text data (only when type is bobText).

Inserts a new object into the current sheet.
The object is inserted at the front of the index specified in index.
If a value lower than 0 is specified in x or y, the insertion position is determined automatically.
If a value lower than 0 is specified in height or width, the object's size is determined automatically.
Do perform the data settings for the IObject object acquired as returned value.

Returned value:
Inserted object
Reference:
Remove()
