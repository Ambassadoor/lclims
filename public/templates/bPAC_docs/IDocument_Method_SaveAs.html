VARIANT_BOOL bpac::IDocument::SaveAs (ExportType type, BSTR filePath)
 

Saves the document as another name specified in filePath

Arguments:
type  The format of the file to be exported.
It is possible to specify bexOpened,bexLbx, and bexLbl.
filePath  Path (UNC) of the file to be saved.
Can be specified as
\\your_server\your_folder\your_file
when saved on a file-sharing server
Returned value:
Success or Failure
