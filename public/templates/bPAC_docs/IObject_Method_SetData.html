VARIANT_BOOL bpac::IObject::SetData (LONG kind, VARIANT data, VARIANT param)
 

Sets the object-specific data

Arguments:
kind  Data kind.
1. Image object
0:Replace the image of image object.
2. Date and time object
0:Set the date and time.
data  Object-specific data to be set.
1. Image object
When kind is 0, it shows the path of the image data to be replaced.
The supported file formats are BMP, non-compressed TIFF, WMF, EMF, ICO, JPEG, DIB, GIF, and PNG.
2. Date and time object
When kind is 0, sets the date or time.
param  Object-specific data parameter to be set.

Object type kind data param
Image 0 TEXT Image file name 0 Adjustable to original image size.
4 Maintains the vertical-horizontal aspect ratio of the image to be replaced.
Date and time 0 DateTime DateTime object NULL No argument

Returned value:
Success or Failure
Reference:
GetData()
