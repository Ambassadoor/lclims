VARIANT_BOOL bpac::IDocument::SetMediaById (LONG mediaId, VARIANT_BOOL fitPage)
 

Sets the media to be used, by media ID

Changes are made to the media of the media ID specified in mediaId.

Arguments:
mediaId  Media ID.
Specify a media ID obtainable from IPrinter::GetSupportedMediaIds().
 
fitPage  As the layout changes due to media changes, specify whether to adjust the size and position of the object in the template in conformity with the changes.
Adjustment is made if VARIANT_TRUE is specified, and no adjustment is made if VARIANT_FALSE is specified.
Returned value:
Success or Failure
Reference:
MediaId
