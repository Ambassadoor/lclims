Using b-PAC with Google Chrome or Microsoft Edge or Mozilla Firefox

To use b-PAC with Chrome or Edge or Firefox you must install the "Brother b-PAC Extention":

  • Chrome

    Install the b-PAC Extension from the Chrome Web Store

  • Edge

    Install the b-PAC Extension from the Edge Add-ons

  • Firefox

    Install the b-PAC Extension stored in the b-PAC SDK folder:
    ...\Redist\Extensions\bpac.xpi


These APIs are promise-based.
