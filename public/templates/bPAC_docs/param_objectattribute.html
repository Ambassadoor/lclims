enum bpac::ObjectAttribute

Object attributes

boaFontBold  (0x1) Font bold

boaFontEffect  (0x2) Font effect

boaFontItalics  (0x3) Font italic

boaFontMaxPoint  (0x4) Maximum point count

boaFontName  (0x5) Font name

boaFontStrikeout  (0x6) Strikeout

boaFontUnderline  (0x7) Font underline

boaDateTimeAddSubtract  (0x8) Addition/subtraction of date and time

boaClipArtGallery  (0x9) Clip art category + No

boaBarcodeProtocol  (0xa) Barcode protocol
