Printout samples and cut option combinations for each major device type

The cut method when printing can be specified.
Various cut options are available and can be specified as follows.

Example If specified by enumerator value beginning with bpo If directly specified by numeric value
Print with only auto cut specified ObjDoc.Open("***.lbx")
ObjDoc.StartPrint "Print sample", bpoAutoCut
ObjDoc.PrintOut 1, bpoAutoCut
ObjDoc.EndPrint
ObjDoc.Close 		ObjDoc.Open("***.lbx")
ObjDoc.StartPrint "Print sample", &H1
ObjDoc.PrintOut 1, &H1
ObjDoc.EndPrint
ObjDoc.Close
Print with auto cut and half cut specified ObjDoc.Open("***.lbx")
ObjDoc.StartPrint "Print sample", (bpoAutoCut Or bpoHalfCut)
ObjDoc.PrintOut 1, (bpoAutoCut Or bpoHalfCut)
ObjDoc.EndPrint
ObjDoc.Close 		ObjDoc.Open("***.lbx")
ObjDoc.StartPrint "Print sample", &H201
ObjDoc.PrintOut 1, &H201
ObjDoc.EndPrint
ObjDoc.Close
